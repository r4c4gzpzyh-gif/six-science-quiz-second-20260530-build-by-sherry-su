# 🔬 小学科学闯关题库

一个完全免费、零成本、无需服务器的小学科学刷题系统。

## ✨ 功能特点

- ✅ **学生登录**：班级、姓名、学号，自动记录学习进度
- ✅ **刷题模式**：顺序刷题、随机练习
- ✅ **错题本**：自动收集错题，随时复习
- ✅ **等级系统**：8个等级，游戏化升级体验
- ✅ **经验值**：做对题目获得经验，激励学习
- ✅ **答案解析**：每道题都有详细解析
- ✅ **打卡记录**：连续做题，培养学习习惯
- ✅ **数据统计**：正确率、做题数一目了然

## 🚀 如何使用

### 方法一：直接在本地打开（最简单）

1. 下载这四个文件到同一个文件夹：
   - `index.html`
   - `style.css`
   - `app.js`
   - `questions.js`

2. 双击打开 `index.html` 即可使用！

### 方法二：部署到 GitHub Pages（免费）

这样所有学生都能通过网址访问：

1. 注册一个 GitHub 账号（免费）
2. 创建一个新仓库
3. 把这四个文件上传到仓库
4. 开启 GitHub Pages 功能
5. 获得一个免费网址，所有人都能访问

详细教程可以搜索"GitHub Pages 部署教程"，5分钟就能搞定。

## 📝 如何添加题目

打开 `questions.js` 文件，按照下面的格式添加：

```javascript
{
    id: 21,                    // 题目编号，按顺序往后加
    type: 'choice',            // 'choice' 单选题，'judge' 判断题
    question: '题目内容在这里',
    options: ['选项A', '选项B', '选项C', '选项D'],  // 判断题只有2个选项
    answer: 0,                 // 正确答案的索引：0=A, 1=B, 2=C, 3=D
    explanation: '这道题的详细解析...',
    difficulty: 2,             // 难度：1-5
    grade: 3,                  // 适用年级：3-6
    category: '植物'            // 知识点分类
}
```

### 添加题目小技巧：

1. **先用 Excel 整理题目**，然后批量转换成 JSON 格式
2. **每道题一定要加解析**，孩子做完能学到东西
3. **题目描述要简洁清楚**，选项不要太长
4. **难度要适中**，从易到难排列

## 📊 数据存储

- 所有数据存在学生的浏览器 `localStorage` 中
- 不需要服务器，不需要数据库
- 清除浏览器数据会丢失记录（建议偶尔导出备份）
- 同一浏览器、同一账号可以接着上次继续做

## 🔧 自定义修改

### 修改等级名称和经验
打开 `app.js`，找到 `LEVELS` 数组，修改即可：

```javascript
const LEVELS = [
    { level: 1, name: '你的等级名称', xp: 0, icon: '🌟' },
    // ...
];
```

### 修改配色
打开 `style.css`，找到第8行的渐变：
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```
换成你喜欢的颜色代码即可。

### 修改标题
打开 `index.html`，第6行的 `<title>` 和第14行的 `<h1>` 都可以改。

## 🆘 常见问题

**Q: 为什么打不开？**
> 确保四个文件都在同一个文件夹里，用 Chrome 或 Edge 浏览器打开。

**Q: 换电脑后数据没了？**
> 是的，数据存在浏览器本地。可以用导出功能备份，或者让学生用同一设备做题。

**Q: 可以放到微信里用吗？**
> 可以！部署到 GitHub Pages 后，把网址发到微信里就能打开。

**Q: 题目可以配图吗？**
> 可以！需要把图片转成 base64 格式，或者上传到图床后用 `<img>` 标签插入题目内容。
